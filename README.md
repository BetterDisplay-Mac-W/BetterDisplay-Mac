## Overview

BetterDisplay Professional Suite is a complete display management platform with color management, HDR control, and multi-display automation for professional display setups.

HDR management provides per-display and per-application HDR settings that go beyond the system-level HDR toggle, enabling fine-tuned HDR behavior for different content types and workflows. The HDR controls handle the common issue of HDR causing unwanted appearance changes in non-HDR content.

Display profiles save complete display configurations — resolution, brightness, color profile, HDR settings, arrangement — and apply them with a single action or automatically based on connected display detection.

Night Shift and True Tone extension brings Apple's blue light reduction and ambient color matching to external displays that do not natively support these features.

BetterDisplay Professional Suite is the display management platform for professionals with multi-monitor setups who need the resolution control, brightness management, and HDR tuning that comprehensive display management requires.

## Performance & Integration

BetterDisplay is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while BetterDisplay processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
